JS的简单内存

栈和堆

  • 对象内容保存在堆中，而其他的属性，比如变量，常量,函数保存在栈中

  • 其实栈中也存在对象，不过只是指向对象的内存地址。

  • 在这个info对象里面保存的friend也是一个指针，指向新的内存地址

  • 所以下面的obj才不会变，因为指向了新的内存地址

  •   值传递
  function foo(a) {
    a = 200
  }
  let num = 100
  foo(num)
  console.log(num) //100

  • 以上两个都是函数里面指向了一个新的内存地址，但原本的地址并没有更改

  • function foo(a) {
      a.name = 'why'
  }

  var obj = {
    name: 'obj'
  }
  foo(obj)
  console.log(obj.name) //why
