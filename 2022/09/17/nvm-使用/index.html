nvm 使用

WHY

  • 通过它可以控制自己node版本。以此方便我们对不同项目node版本的随意切换、安装、查看。。。等等，与npm不同的是，npm是依赖包的管理工具。

注意

  • 我们要先把电脑上已有的node.js删掉。

安装

  • (33条消息) nvm的简介、安装、使用（简单明了）_仰.的博客-CSDN博客_nvm

命令

  • nvm on ：                  打开node.js版本控制
nvm off ：                  关闭node.js版本控制
nvm list ：                  查看已经安装的版本
nvm list available ：   查看网络可以安装的版本
nvm install '版本号' ：安装指定版本node.js
nvm uninstall ' ' ：      卸载指定版本号
nvm use ' ' ：             切换指定的node版本
nvm version ：           查看当前的版本

解决

  • npm 安装报错 npm ERR! Unexpected token ‘.’

  • 该问题多半出现在使用nvm安装node的时候npm会出现错误。

    (此处暂时只提供win系统下的解决办法，其他系统的之后遇到会更重新。)

    首先打开cmd使用nvm卸载npm所属node和npm。

    使用命令 nvm uninstall node版本号 进行卸载。

    点击链接 https://nodejs.org/zh-cn/download/releases/ 找到你所需要的node版本的node压缩包然后下载。

    在你安装nvm时选择的node目录路径下新建文件夹，命名方式为 v+node版本号

  • 然后将下载到的压缩包解压到文件夹里

    打开cmd 输入nvm ls 如果出现了对应版本即为安装成功。

    如果是单纯使用node的话直接卸载node然后再次安装即可解决。
